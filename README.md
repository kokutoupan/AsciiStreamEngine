# AsciiStreamEngine

モダンC++(C++23)で記述された、サーバーサイド・アスキーアート・レンダリング・マルチプレイヤーゲームフレームワークです。
重い行列計算やマルチパス描画（Gバッファ、シャドウマップ等）はすべてサーバー側で実行され、クライアントには最小限の軽量ストリームとして配信されます。

## ディレクトリ構造

- `core/`: フレームワークの心臓部（ネットワークイベントループ、入力解析、描画デバイス基盤）
- `core/shaders/`: シェーダーユーティリティ、アスキーパレット等の共通シェーダー処理
- `client/`: キー入力をEnterなしで即時転送（Rawモード）し、受信フレームを表示するシンクライアント
- `examples/`: フレームワークを利用したサンプルアプリケーション

## Build

ビルドを行うと、ライブラリコア、クライアント、およびサンプルアプリが同時に生成されます。

```sh
rm -rf build # 過去の古いビルドキャッシュがある場合は念のため削除
cmake -S . -B build
cmake --build build

```

## Run

### 1. サーバー（サンプルデモ）の起動

`examples/` にあるマルチプレイヤー同期キューブデモを起動します。

```sh
./build/examples/demo_cube_multiplayer

```

### 2. クライアントの接続

別ターミナルから専用クライアントで接続します。複数立ち上げることで、同一空間内のマルチプレイヤー動作を確認できます。

```sh
./build/client/ascii_client -p 12345

```

- **矢印キー (↑↓←→)**: グローバルな共有ワールドのキューブを回転（全員の画面で同期）
- **W, A, S, D キー**: 自分自身のカメラ視点を移動（自分だけのローカル視点）
- **Space キー**: キューブの回転角を一斉リセット
